# Adriana


Configurar o MySQL:

* Execute o script SQL fornecido
* Ajuste as credenciais na classe Java (URL, USER, PASSWORD)


Adicionar o Driver JDBC:

* Baixe o MySQL Connector/J
* Adicione ao classpath do projeto


Executar:

javac SistemaGerenciamentoAlunos.java
java SistemaGerenciamentoAlunos
